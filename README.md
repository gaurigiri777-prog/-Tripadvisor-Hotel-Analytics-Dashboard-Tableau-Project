# 🌍 Tripadvisor Hotel Analytics Dashboard | Tableau Project

An interactive Tableau dashboard built using Tripadvisor hotel review data to analyze hotel ratings, room capacity, traveler behavior, service availability, and geographical user distribution.

This project focuses on **hospitality analytics, customer review insights, and hotel performance storytelling**.

## 📊 Project Overview

This dashboard helps analyze:

- ⭐ Average hotel ratings
- 🏨 Total hotels by star category
- 🛏 Average number of rooms
- 🌎 Users by continent
- 👥 Traveler type segmentation
- 📅 Period of stay analysis
- 🎯 Top 10 hotels by room capacity
- 🏊 Additional hotel services insights

---

## 🎯 Business Problem

The goal of this dashboard is to help hotel businesses and travel platforms understand:

- customer review behavior
- hotel service trends
- geographical user engagement
- traveler segmentation
- hotel capacity distribution

This helps in improving:
- customer experience
- hotel marketing strategy
- pricing and service planning

---

## 📁 Dataset Used

Tripadvisor hotel review dataset containing:

- User country
- User continent
- Hotel name
- Hotel stars
- Number of rooms
- Review score
- Traveler type
- Period of stay
- Additional services
- Helpful votes
- Number of reviews

---

## 📌 Key KPIs

| KPI | Description |
|------|-------------|
| Avg Score | Average hotel rating score |
| Avg No. of Rooms | Average hotel room capacity |
| Total Hotels | Total number of hotels |
| Users Avg Reviews | Average reviews per user |
